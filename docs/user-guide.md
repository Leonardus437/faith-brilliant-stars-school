## Getting Started

### First Login
1. Navigate to the school portal URL
2. Click "Login" button
3. Enter your email and password
4. You'll be redirected to your dashboard

### Dashboard Overview
The dashboard shows:
- Quick statistics relevant to your role
- Recent activity
- Quick action buttons
- Announcements and notifications

- View report cards
- Pay fees online

### Student
Learning portal:
- View timetable
- Submit assignments
- Check grades
- View announcements

## Common Tasks

### Taking Attendance (Teacher)
1. Go to Attendance page
2. Select your class
3. Select today's date
4. Mark each student as Present, Absent, or Late
5. Click "Save Attendance"
6. Works offline - syncs when online

### Recording Payments (Accountant)
1. Go to Fees & Payments
2. Find the invoice
3. Click "Record Payment"
4. Enter amount and payment method
5. For mobile money: Enter transaction reference
6. Generate receipt

### Viewing Report Cards (Parent)
1. Login to parent portal
2. Select your child
3. Go to "Report Cards"
4. Select term
5. View or download PDF

### Creating Invoices (Accountant)
1. Go to Fees & Payments
2. Click "Create Invoice"
3. Select student and category
4. Enter amount and due date
5. Invoice is auto-generated
6. Parent receives notification

## Mobile Usage

### Optimized for Low-End Devices
- Lightweight pages load quickly
- Large tap targets for easy interaction
- Minimal data usage
- Works on 2G/3G networks

### Mobile Money Payments
1. Select "Pay with Mobile Money"
2. Choose MTN MoMo or Airtel Money
3. Enter phone number
4. Approve on your phone
5. Receive confirmation SMS

### Offline Attendance
1. Open attendance page while online
2. Page caches student list
3. Mark attendance offline
4. Data syncs automatically when online
5. Conflicts resolved by teacher approval

## Offline Mode

### What Works Offline
- View cached student lists
- Take attendance
- View previously loaded data
- Read announcements

### What Requires Internet
- Login/Logout
- Submitting new data
- Viewing updated information
- Making payments
- Downloading reports

### Syncing Data
- Automatic sync when connection restored
- Manual sync button available
- Sync status indicator shows progress
- Conflicts highlighted for review

## Troubleshooting

### Cannot Login
- Check email and password
- Ensure account is active
- Clear browser cache
- Contact admin if issue persists

### Attendance Not Saving
- Check internet connection
- Data saved locally if offline
- Will sync when online
- Check sync status indicator

### Payment Not Reflecting
- Mobile money: Wait 5-10 minutes
- Check transaction reference
- Contact accountant with receipt
- Check spam folder for confirmation email

### Slow Performance
- Clear browser cache
- Close unused tabs
- Check internet speed
- Use recommended browsers (Chrome, Firefox)

### Mobile Money Failed
- Verify phone number
- Ensure sufficient balance
- Check network connection
- Try again after 5 minutes
- Use alternative payment method

### Report Not Generating
- Check if term is completed
- Ensure all grades are entered
- Wait a few seconds for PDF generation
- Try downloading again
- Contact teacher if issue persists
