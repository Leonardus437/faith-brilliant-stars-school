TROUBLESHOOTING
========================================

If parent shows "No Children Found":
- Database was not properly seeded
- Run: python seed_direct.py
- Then run: python fix_passwords.py

If login fails:
- Check password is correct
- Passwords are case-sensitive
- Try copying from this guide

If pages are blank:
- Check backend is running (port 8001)
- Check frontend is running (port 5174)
- Check browser console for errors
