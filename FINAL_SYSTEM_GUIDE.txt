========================================
API ENDPOINTS
========================================

AUTHENTICATION:
POST /api/auth/login - Login

ATTENDANCE (Teacher, Head Teacher):
GET  /api/attendance/classes - Get classes
GET  /api/attendance/students/{class_id} - Get students
POST /api/attendance/submit - Submit attendance
GET  /api/attendance/view/{class_id}/{date} - View attendance
GET  /api/attendance/report/{class_id} - Attendance report

PARENT PORTAL:
GET /api/parent/children - Get parent's children
GET /api/parent/child/{id}/attendance - Child attendance
GET /api/parent/child/{id}/fees - Child fees

STUDENTS (Head Teacher):
GET    /api/students - List students
POST   /api/students - Create student
PUT    /api/students/{id} - Update student
DELETE /api/students/{id} - Delete student

FEES (Accountant, Head Teacher):
GET    /api/fees/invoices - List invoices
POST   /api/fees/invoices - Create invoice
POST   /api/fees/payments - Record payment
DELETE /api/fees/invoices/{id} - Delete invoice

ANALYTICS (All):
GET /api/analytics/dashboard - Dashboard stats
